Sound Records Studio — Site statique prêt à uploader

Contenu :
- index.html : page principale (design sombre, néon)
- logo.png : logo affiché dans le header
- README.txt : ce fichier

Déploiement rapide :
1) Netlify Drop : https://app.netlify.com/drop → Glissez-déposez le dossier ou le ZIP
2) GitHub Pages : repo avec index.html à la racine → Settings → Pages → Branch main → /root
3) Hébergeur au choix : uploadez tout le dossier à la racine du site

Personnalisation :
- Dans index.html, remplacez les liens des boutons "Acheter" et "Écouter complet" par vos URLs Gumroad / SoundCloud / BeatStars.
- Remplacez aussi le lien TikTok (#) par votre URL TikTok.
- Dupliquez le bloc .track pour ajouter des nouvelles instrus.

Bonnes ventes !
